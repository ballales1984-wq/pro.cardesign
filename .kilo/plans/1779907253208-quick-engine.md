## Available Features

### voxelToMesh Options
```javascript
voxelToMesh(voxels, {
  voxelSize: 1,        // mm per voxel
  flatCubes: true,     // Face-culled cubes (external only)
  wireframe: true,     // All edges for internal view
  faceSubdivisions: 2,  // Grid density per face
  isoValue: 0.5        // MC surface threshold
})
```

### VoxelEngine Methods
```javascript
renderer.update(deltaTime);  // Auto-updates LOD
engine.toggleSurfaceMesh();  // Toggle MC surface
engine.toggleWireframe();    // Toggle internal edges
```

### VideoKeyframeExtraction
```javascript
const vke = new VideoKeyframeExtraction(voxelEngine, proceduralEngine);
const keyframes = await vke.extractKeyframes(videoFile, { interval: 30, sensitivity: 0.3 });
const parallel = await vke.extractKeyframesParallel(videoFile, { regions: 4 });
vke.play((frame, t) => { /* render frame */ }, 30);
vke.seekTo(5.0);  // Jump to 5 second mark
```
