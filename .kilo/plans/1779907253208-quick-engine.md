# Pro.Cardesign Implementation Plan

## Overview
Analysis of in-progress phases (Phase 2, Phase 4, Phase 7, Phase 8) and implementation roadmap.

---

## Phase 2: VISUALIZATION (In Progress)

### Current Status
- ✅ `voxelToMesh.js` - Has `_flatCubes` path with face-culling for external surface extraction
- ✅ `LODManager` - Distance-based detail management implemented
- ⏳ **Missing**: Optional internal wireframe, automatic LOD integration

### Tasks
1. **External Surface Extraction** (`src/geometry/converters/voxelToMesh.js:90-96`)
   - The `_surfaceCubes` function currently calls `_flatCubes` - needs true external-only extraction
   - Enhancement: Add `extractExternalSurface()` method that only outputs faces with no voxel neighbors

2. **Internal Wireframe Option**
   - Add `wireframeMode` option to voxelToMesh
   - When enabled, render edges-only for internal voxels (non-exposed faces)
   - Toggle via UI button

3. **Automatic LOD Integration**
   - Connect `LODManager` to `VoxelEngine.update()` loop
   - Apply LOD levels to instance visibility
